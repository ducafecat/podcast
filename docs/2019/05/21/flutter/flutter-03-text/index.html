1.2 构造函数

1.3 构造参数表

名称 说明
data 文本
style 样式
strutStyle 支柱风格
textAlign 对齐方式
textDirection 文本方向
locale 语言环境
softWrap 自动换行
overflow 文字溢出处理方式
textScaleFactor 字体缩放
maxLines 最大显示行数
semanticsLabel 图像描述

1.4 style 样式参数表

名称 说明
inherit = true 是否继承父类组件属性
color 字体颜色
backgroundColor 背景颜色
fontSize 文字大小，默认 14px
fontWeight 字体粗细
fontStyle 字体样式
letterSpacing 字母间距
wordSpacing 单词间距
textBaseline 字体基线
height 行高
locale 设置区域
foreground 前景色
background 背景色
shadows 阴影
decoration 文字划线
decorationColor 划线颜色
decorationStyle 划线样式，虚线、实线等样式
decorationThickness 划线厚度
debugLabel 描述信息
String fontFamily 字体
