一些优秀的路由插件

  • fluro

    前端的使用体验

    router.navigateTo(context, “/users/1234”, transition: TransitionType.fadeIn);

  • flutter_modular

    功能强大的路由管理：中间件、懒加载、状态管理、动态路由、分组路由、动画、返回值、命名路由

  • auto_route

    设计精简、低耦合其它功能

    功能：中间件、自动生成路由代码、动态路由、动画、返回值、命名路由
