将要使用的软件包:

  • Chopper:
    使用 source gen 和 Retrofit 的 Dart 和 Flutter 的 Http 客户端生成器

  • Json Serializable:
    通过使用它，将自动生成实体的 JSON 转换

  • Japx:
    解析器，它将复杂的[ JSON: API ][1]结构变为简单的 JSON，反之亦然

非常感谢这些令人敬畏的包裹！
